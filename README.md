# 基于 uni-app & ElementUI 的后台管理项目模板

> 真正使用 HBuilderX 开发管理后台，兼容PC、移动端。不需要 uniCloud

### 初始化

```bash
git clone https://gitee.com/mc-uniapp/uni-admin.git

cd uni-admin

npm install
# or
yarn install
```

### 运行

直接在 HBuilderX 中点击 运行 > 运行到浏览器
